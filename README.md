# Voidmap

Находи и обрабатывай сигналы из космоса. Вдохновлено Voices of the Void.

## Установка и запуск
``` bash
git clone https://github.com/iwakura-X/voidmap.git
cd voidmap
python main.py
```
Никаких дополнительных модулей и библиотек, кроме самого Python не требуется.

## Команды

| Команда | Описание |
|---------|----------|
| `tune <частота> <R/L>` | Настроить телескоп, например `tune 1420 R` |
| `scan` | Сканировать эфир (3 секунды) |
| `list` | Показать найденные (ещё не сохранённые) сигналы |
| `save <номер>` | Сохранить сигнал в каталог |
| `catalog` | Показать все сохранённые сигналы |
| `process <номер>` | Повысить уровень обработки сигнала (открывает новые данные) |
| `ping` | Радар – примерная наводка на ближайший неизученный сигнал |
| `exp` | Прогресс: сколько сигналов обработано и доступный уровень |
| `q` | Выйти из игры |

## Прогрессия

- Обработал **3 разных сигнала** → открывается уровень обработки 2
- Обработал **10 разных сигналов** → открывается уровень обработки 3

Чем выше уровень, тем больше деталей о сигнале.

## Формат базы сигналов (`signals_db.json`)

```json
{
    "Название источника": {
        "freq": 1420.0,
        "stren": 0.8,
        "pol": "R",
        "info_levels": {
            "1": "Текст после идентификации",
            "2": "Дополнительные данные",
            "3": "Секретная информация"
        }
    }
}
```
